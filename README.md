# Sistema de Gestión de Servicios

Aplicación de consola en Python para gestionar servicios, especialistas y reservas.

## Descripción
El sistema permite a un empleado administrar:
- servicios
- especialistas
- reservas

La aplicación inicia directamente en el menú principal del empleado, sin pantalla de bienvenida adicional.

## Características
- CRUD completo para servicios, especialistas y reservas
- Validaciones de disponibilidad y restricciones de eliminación
- Datos en memoria durante la sesión
- Interfaz por consola

## Requisitos
- Python 3.8+
- No requiere dependencias externas

## Ejecución
```bash
python src/main.py
```
